# @stacktape/cli-<<PLATFORM>>

This package contains the Stacktape CLI binary for <<PLATFORM_DISPLAY>>.

**You don't need to install this package directly.** It's automatically installed as an optional dependency when you install the main `stacktape` package.

## Installation

Install the main Stacktape package instead:

```bash
npm install -g stacktape
```

or

```bash
npx stacktape --help
```
