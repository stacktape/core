# Compute Resources API Reference

This section contains detailed API reference documentation for all compute resources available in Stacktape.

## Available Resources

- [Web Service](./api-reference/web-service.mdx) - Continuously running container with public endpoint
- [Private Service](./api-reference/private-service.mdx) - Continuously running container with private endpoint
- [Worker Service](./api-reference/worker-service.mdx) - Continuously running container not accessible from outside
- [Multi-Container Workload](./api-reference/multi-container-workload.mdx) - Custom multi-container workload
- [Batch Job](./api-reference/batch-job.mdx) - Simple container job that runs and terminates
- [Lambda Function](./api-reference/function.mdx) - Serverless function
- [Edge Lambda Function](./api-reference/edge-lambda-function.mdx) - Lambda function running at CloudFront edge locations
- [Next.js Web](./api-reference/nextjs-web.mdx) - Next.js application deployment

Each resource page contains comprehensive configuration examples, property descriptions, and union type alternatives.
